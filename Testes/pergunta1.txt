Pergunta 1 teste 1

## 1 Ponto

Qual valor terá a variável de resultado após executar o código a seguir em Python?

```
número = 7
se número % 2 == 0
           resultado = "Par"
señao:
           resultado = "Impar"
```

Grupo de escolhas da pergunta

- [ ]  a) “par”
- [x]  b) “Impar”
- [ ]  c) 7
- [ ]  d) Erro de sintaxe (TypeError)

---

## **1. Entendendo o código**

O enunciado em “português” seria algo assim (em Python correto):

```python
numero = 7
if numero % 2 == 0:
    resultado = "Par"
else:
    resultado = "Impar"
```

A ideia é:

- `numero = 7` define a variável com valor 7.
- `numero % 2` calcula o resto da divisão de `numero` por 2. Se o resto for 0, o número é par; se não for 0, é ímpar.

## **2. Qual condição é verdadeira?**

A condição testada é:

`numero % 2 == 0`

- Se for verdadeira, `resultado` recebe `"Par"`.
- Se for falsa, cai no `else` e `resultado` recebe `"Impar"`.

Como `numero = 7`, precisamos decidir se `7 % 2` é igual a 0 ou não.

## **3. Hora de pensar**

Agora Pense:

- Quanto vale `7 % 2`?
- Esse resultado é igual a 0 ou não?
- Com isso, qual dos blocos será executado: o que atribui `"Par"` ou o que atribui `"Impar"`?

**Resposta: b) “Impar”**

conferir a lógica:

- `numero = 7`
- `numero % 2` calcula a divisão de 7 por 2. Como 7 é ímpar, o resto é 1, não 0.
- A condição `numero % 2 == 0` é **falsa**, então o bloco do `if` não roda.
- O fluxo vai para o `else`, atribuindo `resultado = "Impar"`.

Então no final `resultado` guarda `"Impar"`, que corresponde à alternativa **b**.
